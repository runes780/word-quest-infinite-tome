# Image Generation Prompts

These prompts are designed for README and open-source project visuals. Use synthetic visuals only. Do not include real students, school logos, copyrighted characters, or identifiable personal data.

## 1. Hero Banner

- Intended file name: `docs/assets/hero.png`
- Aspect ratio: 16:9
- Style: polished educational fantasy game UI concept art
- Color palette: deep blue, gold, soft purple
- Text requirement: no readable or unreadable text, no watermark

```text
Create a 16:9 polished educational fantasy game UI concept art scene for an open-source vocabulary learning project. A young learner avatar explores an infinite magical tome in a warm starry library. Blank vocabulary cards float around the open book, glowing gold learning paths connect small islands of knowledge, and friendly non-scary fantasy companions watch nearby. Use deep blue, gold, and soft purple. No real people, no copyrighted characters, no school logos, no text, no watermark.
```
